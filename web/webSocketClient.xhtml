WebSockets API

Tato stránka slouží ke zkoušení komunikace websockety. Budou zde uvedeny příkazy implementované na serveru, které lze použít použít pro komunikaci a server s nimi bude nakládat rozdílně.

Dále zde budou uvedena schémata pro JSON objekty, které využijeme pro jednotlivé typy komunikací.

Příkaz Popis
getDB Vypíše veškerý obsah z databáze.
getDB((ASC)|(DESC),POČET_ŘÁDKŮ) Vypíše definovaný počet řádků obsah z databáze ASC = vzestupně, DESC = sestupně.
addUser(JMÉNO,PŘÍJMENÍ) Přidá uživatele do databáze.
getUser(ČÍSLO) Vrátí uživatele se specifikovaným ID.
